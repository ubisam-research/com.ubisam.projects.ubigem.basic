2. 프로젝트 빌드
    1> **솔루션탐색기** 열기
    2> **UbiGEM.Net** 프로젝트 마우스 오른쪽 클릭하고 팝업메뉴에서 **다시빌드**
    
# Confused 파일 복사
1. **..\BIN\Confused\**  폴더로 이동
2. **UbiGEM.Net & UbiCOM.Nt** 파일 복사
3. **..\Setup\BIN Data** 폴더에 붙혀넣기

# build msi 
1. **..\Setup\UbiGEM\** 폴더로 이동
2. **UbiGEM.sln** 프로젝트 열기
3. build msi 
    1> **솔루션탐색기** 열기
    2> **UbiGEM** 프로젝트 마우스 오른쪽 클릭하고 팝업메뉴에서 **다시빌드**
    3> **..\Setup\Setup Files\** 폴더로 이동.
    4> **UbiGEM.msi** 파일 압축하여 배포.
